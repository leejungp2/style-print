# StylePrint Wiki

StylePrint는 UI 레퍼런스 스크린샷에서 디자인 특징을 추출하고, 여러 레퍼런스의 스타일 요소를 조합해 React + Tailwind UI 코드를 생성하는 서비스입니다.

## 1주차 산출물

- [1주차 프로젝트 기획서](StylePrint-1주차-프로젝트-기획서)

## 운영 원칙

- 개발 Task는 GitHub Issue로 관리한다.
- 기획서, 주차별 산출물, workflow 문서는 GitHub Wiki로 관리한다.
- branch는 `main` -> `dev` -> `feature/*` 흐름으로 관리한다.
- 기능 개발은 `feature/*` branch에서 진행하고 `dev` branch로 PR을 생성한다.
